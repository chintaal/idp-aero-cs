## Recommended Additions for a Stronger Report

- recent RocketPy implementation references or documentation for reproducibility,
- sensor datasheets for the chosen barometer and IMU,
- servo datasheet for torque and speed assumptions,
- and any competition or range-safety documents governing the intended test environment.

## Citation Advice

When converting the project into a formal report or paper:

- keep one citation style consistently,
- separate foundational textbooks from recent application papers,
- and cite simulator assumptions, not just algorithms.
